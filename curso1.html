CURSO 1

Instalación de sistema operativo

Para tener un entorno de trabajo ideal, para poder realizar funciones de pentesting, es el BSPWM. Este entorno necesitará de un sistema operativo que lo respalde, desde aqui se tiene que hacer en una distribucion de linux. Nuestras recomendaciones són: Kali Linux i Parrot OS.

Entorno BSPWM

Este entorno esta caracterizado por su sencillez y rapidez a la hora de abrir y cerrar ventanas. Se respalda principalmente en SXHKD para los "shorcuts". Con una shell de ZSHELL, se personaliza para un mayor grado de conformidad a la hora de trabajar.
